=====================================
Welcome to detectCOs's documentation!
=====================================


**detectCOs** is a Python tool designed for detecting crossover positions from genotype-by-sequencing data. For more information about chromosomal crossovers, refer to `Wikipedia <https://en.wikipedia.org/wiki/Chromosomal_crossover>`_. 

.. note::

   This project is under active development.
   
Check out the :doc:`usage` section for further information, including instructions on how to :ref:`install <installation>` and :ref:`initiate <launch>` the project.

Overview
--------

This documentation covers additional functionalities developed for the detectCOs program, which aims to enhance our ability to analyze and interpret crossover events in genetic studies. These enhancements were contributed by a team of dedicated researchers and programmers who have carefully corrected and improved upon the initial release.
